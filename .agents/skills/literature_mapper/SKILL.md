# Literature Mapper Skill

## Where to work
- `docs/literature/INDEX.md`
- `docs/literature/*.md`
- `paper/refs.bib`

## Output format
For each entry:
- Citation key
- Tags (survey / heuristics / computation / dynamics / visualization / etc.)
- Summary (<= 120 words)
- Relevance to this repo (1-2 bullets)
- Where cited (`paper/sections/...`)
