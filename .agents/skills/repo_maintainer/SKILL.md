# Repo Maintainer Skill

## Where to work
- Root files (`README.md`, `.gitignore`, `pyproject.toml`)
- `.github/` workflows and templates
- `tests/`

## Rules
- Prefer small refactors with clear rationale.
- Add/adjust tests when changing behavior.
- Keep scripts runnable on Windows + Linux when feasible.
