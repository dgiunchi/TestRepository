# Related Work Skill

## Where to work
- `paper/sections/related_work.tex`
- `paper/refs.bib`
- `docs/literature/` (create notes)

## Process
1. Create a **taxonomy** (surveys, stopping time statistics, random-walk heuristics, generalized maps, computational verification, visualization).
2. For each cited work, add:
   - 2-4 sentence summary
   - what we reuse / what differs
   - bib entry key
3. Keep claims verifiable; avoid asserting results without a citation.

## Deliverables
- A structured related-work section
- Bib file updated with consistent keys
