# Idea Lab Skill

## Where to work
- `docs/ideas/queue.md`
- `docs/ideas/<slug>.md`
- `analysis/` or `src/` only to scaffold experiments (if asked)

## Rules
- Each idea must include:
  - Hypothesis statement
  - Proposed metric / invariant
  - Minimal experiment to test it
  - Failure modes / what would disconfirm it
- Prefer ideas that yield **new plots** or **new distributions**.
