\section{Introduction}

The Collatz map is defined by iterating the rule ``halve if even, else $3n+1$''.
Despite its elementary formulation, the global behavior of trajectories remains mysterious.

\paragraph{Scope of this work.}
This repository focuses on \emph{reproducible computation} and \emph{visual explanation}. We aim to:
\begin{itemize}
  \item provide baseline datasets and plots for $\sigma(n)$ and $\pi(n)$ over large ranges,
  \item develop new visual encodings for trajectory structure,
  \item formulate and test computationally-accessible hypotheses.
\end{itemize}
