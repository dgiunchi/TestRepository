Core definition

The Collatz map is $T(n)=n/2$ if $n$ is even, and $T(n)=3n+1$ if $n$ is odd.

Example code

def collatz_next(n: int) -> int:
    return n // 2 if n % 2 == 0 else 3 * n + 1
